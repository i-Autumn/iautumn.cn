基础命令

Router(config)#host 主机名 //设置交换机的主机名
Router(config)#write //保存配置
Router(config)#(Ctrl+z) //返回特权模式
Router(config)#int 接口 //进入接口
router(config-if)#no shutdown //激活当前接口

进阶操作

router(config)#enable secret xxx 设置特权加密口令
router(config)#enable password xxb 设置特权非密口令
router(config)#line console 0 进入控制台口
router(config-line)#line vty 0 4 进入虚拟终端
router(config-line)#login 要求口令验证
router(config-line)#password xx 设置登录口令xx

静态路由

  • 静态路由是指网络管理员手动配置的路由信息。手动配置静态路由需要网络管理员非常了解网络拓扑结构，在网络规模较大时还需耗费大量的精力和时间，所以不是所有网络都适合使用静态路由。
  • 默认路由是一种特殊的静态路由，当路由表中没有对应于目标网络的路由条目时，路由器将使用默认路由转发数据。一般情况下，当网络中存在末梢网络时使用默认路由。这是因为末端网络通常只有一条出口路径，使用默认路由可以简化路由器配置，提高效率。

动态路由

  • 动态路由是基于某种协议实现的，常见的路由协议有内部网关协议（IGP）和边界网关协议（BGP），其中内部网关协议又分为路由信息协议（RIP）和开放式最短路径优先（OSPF）协议。

RIP路由

  • RIP（Routing Information Protocol，路由信息协议）是最先得到广泛应用的内部网关路由协议，它使用距离矢量来决定数据到达目的地的最佳路径
    RIP允许的跳数最大为15跳，超过15跳的网络将无法到达，因此RIP一般适用于规模较小的同构网络。

    OSPF路由

  • OSPF（Open Shortest Path First）用于在单一自治系统（AS，Autonomous System）内决策路由。OSPF协议是基于链路状态的，因此也可称为链路状态协议。

路由重分发

  • 在大型的企业中，可能在同一网内使用到多种路由协议，为了实现多种路由协议的协同工作，路由器可以使用路由重分发（route redistribution）将其学习到的一种路由协议的路由通过另一种路由协议广播出去，这样网络的所有部分都可以连通了。 为了实现重分发，路由器必须同时运行多种路由协议，这样，每种路由协议才可以取路由表中的所有或部分其他协议的路由来进行广播。

单臂路由

  • 单臂路由（router-on-a-stick）是指在路由器的一个接口上通过配置子接口（或“逻辑接口”，并不存在真正物理接口）的方式，实现原来相互隔离的不同VLAN（虚拟局域网）之间的互联互通。
    在Cisco网络认证体系中，单臂路由是一个重要的学习知识点。通过单臂路由的学习，能够深入的了解VLAN（虚拟局域网）的划分、封装和通信原理，理解路由器子接口、ISL协议和802.1Q协议，是CCNA考试中经常考的。

访问控制列表(ACL)

  • 访问控制列表（ACL，Access Control Lists）是应用在路由器（或三层交换机）接口上的指令列表，用来告诉路由器哪些数据可以接收，哪些数据是需要被拒绝并丢弃。

标准ACL

  • 标准ACL只能通过源地址进行访问过滤，因此只能阻止/允许来自指定IP地址的访问。
    将标准ACL放置在靠近目标的位置

扩展ACL

  • 和标准ACL相比，扩展ACL拥有更加强大的功能，除了可以使用源地址进行匹配，还可以使用目的地址、端口号、协议等进行匹配，从而实现对访问的精确控制。

NAT协议

  • NAT（Network Address Translation），译为网络地址转换，是一个Internet工程任务组标准。它可以实现内部私有IP地址和公网IP地址的转换，起到节约公网IP地址的作用。

  • 公网IP地址是指在Internet中使用的地址，每个公网地址在全球都是唯一的。私有IP地址是一段保留地址，无法在Internet中使用，只能在局域网内部自由使用。A、B、C三类IP地址中分别预留出了一段地址范围用作私有地址。

  • 四中地址说明：
    ①内部本地地址：指局域网内部分配给主机的IP地址，通常为私有地址。
    ②内部全局地址：代表内部网络到外部网络的公网IP地址，一般设置在路由器等互联设备上，用于外部网络中的路由。
    ③外部本地地址：指互联网上另一端网络内部的IP地址，该IP地址可能是公网地址，也可能是私有地址。
    ④指互联网上的公网IP地址，一般情况下，外部本地地址和外部全局地址是同一个地址，它们内部网络所访问的互联网上的主机。

静态NAT

动态NAT

  • 动态NAT是指内部本地地址与内部全局地址进行转换时，内部私有地址可以随机转换为任何指定的全局地址。也就是说，指定的内部私有地址和合法的内部全局地址之间可以实现动态转换，不需要一一设置映射，从而减少了配置的工作量。

PAT

  • 端口多路复用即端口地址转换（PAT，Port Address Translation），是指将内部本地IP地址动态地转换为单一的内部全局IP地址和端口号。

动态NAPT

  • NAT是Network Address Translation(网络地址转换)的缩略语，是使用一台路由器在你的专用网络中的PC之间实现互连网接入共享的一种技术，尽管那些PC没有一个合法的公共IP地址。
  • NAT主要分为两种类型：NAT和NAPT。NAT是实现网络地址转换后一个本地IP地址对应一个全局地址。NAPT是实现多个本地IP地址对应一个全局地址。
