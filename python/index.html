运算符

Python 语言支持以下类型的运算符:
算数运算符 +(加)、-(减)、*(乘)、/(除)、%(取模,返回余数)、**(幂)、//(取整数)
比较(关系)运算符 ==、!=、>、<、>=、<=、
赋值运算符 =、-=、+=、*=、/*、%=、**=、//=、:=(海象运算符，可在表达式内部为变量赋值)
逻辑运算符 and、or、not
位运算符 暂时不做笔记,后面再补...
成员运算符 in、not in
身份运算符 is(判断两个标识符是不是引用自一个对象)、is not(是判断两个标识符是不是引用自不同对象)

  • 运算符优先级(同级按照正常逻辑思维)
    算数运算符>>>位运算符>>>比较运算符>>>赋值运算符>>>身份运算符>>>成员运算符>>>逻辑运算符

控制流语句

if条件控制

if number_name1:
    start_01
elif number_name2:
    satrt_02
else:
    start_03

注意：
1、每个条件后面要使用冒号 :，表示接下来是满足条件后要执行的语句块。
2、使用缩进来划分语句块，相同缩进数的语句在一起组成一个语句块。
3、在Python中没有switch – case语句。
